# CoronaApp
Sample App using [BaseMVVM](https://github.com/ttmagic/BaseMVVM), showing news related to Corona disease and its statistic in Vietnam.

Language: Kotlin

Architecture: MVVM https://developer.android.com/jetpack/docs/guide
  + Data binding
  + ViewModel + LiveData
  + Jetpack Navigation
  + Kotlin Coroutines
